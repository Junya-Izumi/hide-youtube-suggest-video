# hide-youtube-suggest-video
Youtubeを見ているときに右側に出てくる関連動画を非表示にするChrome拡張機能です<br>
cssで対象を`display: none;`するだけのシンプルな仕組みでできており、とても軽量です
## 作った理由・目的
私は普段、見たいものや調べたいことがあってyoutubeを見ていますが、興味が湧いてくる関連動画をついついクリックしてしまい動画を必要以上に見てしまっています<br>
そんな時間の浪費を防止しようと思い、この拡張機能を作りました
## インストール方法
1. このリポジトリを clone または ZIPをダウンロードして展開する
```
git clone https://github.com/Junya-Izumi/hide-youtube-suggest-video.git
```
2. Chrome で chrome://extensions を開く
3. 画面の右上にある「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能読み込む」をクリックし、ダウンロードしたフォルダを選択する
